## Overview
This project analyzes GitHub repositories and generates:
- A score based on project quality
- A summary of the repository
- A roadmap for improvement

## How it Works
1. Enter a GitHub repo URL in the app.
2. `github_fetcher.py` fetches repo info.
3. `scorer.py` calculates score and summary.
4. `roadmap.py` generates recommended improvements.
